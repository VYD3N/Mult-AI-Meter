# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/0ac3104e-48c4-484b-8a3e-88588f71bf86

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `OPENROUTER_API_KEY` in `.env.local`
3. Optional: set `OPENROUTER_MODEL` to override the default model (`google/gemma-3-27b-it:free`)
4. Run the app:
   `npm run dev`
